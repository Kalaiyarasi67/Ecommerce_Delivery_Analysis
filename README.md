**Ecommerce Delivery Analysis Dashboard
**
Project Overview

The Ecommerce Delivery Analysis Dashboard is a Power BI project designed to analyze delivery performance, customer satisfaction, refund trends, and product category insights across multiple ecommerce delivery platforms such as Blinkit, JioMart, and Swiggy Instamart.

This dashboard helps businesses monitor operational efficiency and improve customer experience through interactive visual analytics.


---

Objective

The main objective of this project is to:

Analyze ecommerce delivery performance.

Compare platforms based on service quality.

Track delivery delays and refund requests.

Understand customer feedback patterns.

Identify product categories with higher delivery issues.



---

Tools & Technologies Used

Power BI

Microsoft Excel

Data Cleaning & Transformation

Data Visualization



---

Dataset Information

The dataset contains ecommerce delivery records with the following fields:

Column Name	Description

Order ID	Unique order identifier
Customer ID	Unique customer identifier
Platform	Delivery platform name
Delivery Time (Minutes)	Total delivery duration
Product Category	Category of ordered products
Order Value (INR)	Total order amount
Customer Feedback	Customer review/comments
Service Rating	Rating provided by customer
Delivery Delay	Indicates delayed delivery
Refund Requested	Indicates whether refund was requested

Key Insights

Platforms with lower delivery time generally received higher service ratings.

Delayed deliveries increased refund requests.

Personal Care and Grocery categories generated higher order values.

Some platforms showed higher customer satisfaction despite delivery variations.

Refund trends highlighted operational inefficiencies in specific platforms.
